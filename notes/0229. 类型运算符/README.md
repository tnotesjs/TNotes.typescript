## 2. 🫧 评价

本章主要介绍 TS 中类型运算的相关知识。

TS 中的很多内置的工具类型，比如 `Partial<T>`、`Required<T>`、`Readonly<T>`、`Record<T,U>`、`Pick<T,U>`、`Exclude<T,U>` …… 等等，都是基于这些基础的运算逻辑实现的。
