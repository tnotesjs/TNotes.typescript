## 1. 🎯 本节内容

- 字符串枚举 vs 数字枚举

## 2. 🫧 评价

这篇笔记主要介绍字符串枚举和数字枚举的差异对比，两者的核心差异在于是否支持自动初始化以及反向映射。

## 3. 🆚 字符串枚举 vs 数字枚举

| 特性     | 字符串枚举 | 数字枚举 |
| -------- | ---------- | -------- |
| 值类型   | 字符串     | 数字     |
| 自动递增 | 不支持     | 支持     |
| 反向映射 | 不支持     | 支持     |

因为字符串枚举无法自动自增，因此必须显示初始化。
