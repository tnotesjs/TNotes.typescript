## 3. 🤔 string 类型是什么？

- 在 TypeScript 中，string 是一种基本数据类型，用于表示文本数据。
- string 类型包含所有字符串，普通字符串和模板字符串都属于 string 类型。

```ts
const x: string = 'hello'
const y: string = `${x} world`
```

上面示例中，普通字符串和模板字符串都属于 string 类型。
